# LearnPlaywrightFundamentals2X

This repository contains a Playwright automation project for learning Playwright fundamentals.

## Project Structure

- `package.json` - Node.js package configuration
- `playwright.config.ts` - Playwright configuration file
- `tests/` - Test files for Playwright
- `node_modules/` - Installed dependencies (excluded from Git)

## Getting Started

1. Install dependencies:

```bash
npm install
```

2. Run Playwright tests:

```bash
npx playwright test
```

## Notes

- The `node_modules` folder is excluded from version control using `.gitignore`.
- This repository is intended for learning Playwright automation basics.
